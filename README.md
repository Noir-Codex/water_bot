# 💧 Умный Telegram-бот для Трекинга Воды (WaterBot)

[![Python](https://img.shields.io/badge/Python-3.13+-blue.svg)](https://www.python.org/)
[![aiogram](https://img.shields.io/badge/aiogram-3.x-green.svg)](https://docs.aiogram.dev/en/latest/)
[![SQLite](https://img.shields.io/badge/SQLite-aiosqlite-lightgrey.svg)](https://docs.python.org/3/library/sqlite3.html)

WaterBot — это асинхронный Telegram-бот, который помогает поддерживать правильный водный баланс. В отличие от других трекеров, он **не спамит** уведомлениями каждый час. Бот использует "умный" алгоритм: он анализирует ваши активные часы и присылает напоминание *только* в том случае, если вы отстаёте от своего идеального графика потребления воды.

---

## ✨ Ключевые возможности

- 🎯 **Индивидуальная норма**: Автоматически рассчитывает суточную норму воды исходя из вашего веса по формуле ВОЗ (32 мл на 1 кг).
- 🧠 **Умные напоминания**: Учитывает время вашего пробуждения и сна. Отслеживает текущий прогресс и напоминает попить только при отставании от графика, выбирая разные тексты для разнообразия.
- ⚡ **Быстрый ввод**: Удобные инлайн-кнопки для добавления воды в один клик (150, 250, 350, 500 мл), а также возможность ввести **свой собственный объем**.
- 📊 **Наглядная статистика**: Красивые текстовые прогресс-бары, мотивирующие статусы и отслеживание выполнения дневной цели.
- ⚙️ **Гибкие настройки**: Встроенный профиль с возможностью в любой момент изменить цель, вес или часы сна.

## 🛠 Технологический стек

- **Язык**: Python 3.13+
- **Фреймворк**: [aiogram 3.x](https://docs.aiogram.dev/) (для асинхронной работы с Telegram API)
- **База данных**: SQLite (асинхронный доступ через `aiosqlite`)
- **Планировщик задач**: [APScheduler](https://apscheduler.readthedocs.io/) (интервальные проверки прогресса в фоне)
- **Управление окружением**: `python-dotenv`

## 🚀 Установка и запуск

### 1. Подготовка
Склонируйте репозиторий:
```bash
git clone https://github.com/Noir-Codex/water_bot.git
cd water_bot
```

### 2. Настройка виртуального окружения
Создайте и активируйте виртуальное окружение, чтобы изолировать зависимости проекта:
```bash
python3 -m venv venv
source venv/bin/activate  # Для macOS/Linux
# venv\Scripts\activate   # Для Windows
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Настройка переменных окружения
Скопируйте пример файла конфигурации:
```bash
cp .env.example .env
```
Затем заполните `.env` своими данными (нажмите на ✏️ или откройте в любом редакторе):
```env
# Ваш токен от @BotFather в Telegram:
BOT_TOKEN=1234567890:ABCdefGhIJKLmnopQRstUVwxyZ
# Файл локальной базы данных SQLite:
DB_PATH=water_bot.db
```

### 5. Запуск
Во время работы в виртуальном окружении запустите бота:
```bash
python main.py
```
В консоли появится сообщение: `[INFO] __main__: Бот запущен!` 🎉

## 💡 Как работает "Умный" планировщик (`scheduler.py`)

Каждые **45 минут** `APScheduler` активирует алгоритм проверки для каждого пользователя:
1. Запрашивает текущее время и сверяет с установленными "активными часами" (от подъема до сна).
2. Вычисляет *идеальное ("ожидаемое") количество воды* на данный момент времени через линейную интерполяцию.
3. Дополнительно проверяет отсутствие недавних (менее 90 минут назад) напоминаний.
4. Если выпитый объем составляет **менее 80%** от ожидаемого на текущую минуту — бот присылает напоминание.

Такой подход защищает пользователя от спама, если он выпивает свою норму в срок.
